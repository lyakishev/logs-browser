Данные для окна просмотра лога

Данные для окна просмотра лога берутся из результата выполнения следующего запроса:

		 select date, logname, type, source, pretty(log), lid 
                 from $this where lid in идентификаторы_сообщений
	         order by date asc, lid desc

, где идентификаторы_сообщений - значение скрытой колонки rows_for_log_window, в которой хранятся идентификаторы всех сообщений, соответствующих данной строке.
